# Life Care 照護系統

護理之家照護系統原型，包含登入、線上打卡、後台打卡紀錄、生理照護與生命徵象紀錄。

## 本機啟動

```bash
npm start
```

啟動後開啟：

```text
http://localhost:3000
```

## 資料儲存

用 `npm start` 開啟網站時，打卡紀錄與生命徵象會儲存在：

```text
data/db.json
```

如果直接用檔案方式開啟 `index.html`，系統會退回使用瀏覽器本機儲存。

## 目前功能

- 帳號、密碼、職位登入
- 護理師、照服員、主管分開打卡
- 後台查看打卡紀錄
- 生理模組
- 生命徵象住民清單、展開今日紀錄
- 批次輸入生命徵象
- 異常生命徵象紅色提示與儲存確認
